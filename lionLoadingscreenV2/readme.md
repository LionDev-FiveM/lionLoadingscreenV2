[Installation]
1. Drag & Drop the Script of your server
2. Open your Server.cfg / CFG Editor in TxAdmin
3. Add "ensure lionLoadingscreenV2" in your cfg
4. Go to index.html.lua
5. Here you can edit all
6. Enjoy it!
